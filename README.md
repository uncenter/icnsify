# icnsify

Convert PNGs to [`.icns`](https://en.wikipedia.org/wiki/Apple_Icon_Image_format).

## Installation

### Cargo

```sh
cargo install icnsify
```

### Nix

[Available through Nixpkgs](https://nixpkgs.dev/icnsify).

```
nix run nixpkgs#icnsify
```

## Usage

```
icnsify [--output] <INPUT>
```
